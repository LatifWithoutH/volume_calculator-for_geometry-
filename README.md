# 📐 Kalkulator Volume Bangun Ruang (Flutter)
Aplikasi mobile sederhana dan modern yang dibangun menggunakan Flutter untuk menghitung volume berbagai bangun ruang dengan cepat dan akurat. Dirancang dengan antarmuka yang ramah pengguna (user-friendly) dan mendukung multi-platform (Android, iOS, Web, Windows).

✨ Fitur Utama
🔲 Kubus: Hitung volume berdasarkan panjang sisi.
🛢️ Tabung: Hitung volume berdasarkan radius dan tinggi.
⚽ Bola: Hitung volume berdasarkan radius.
🎨 UI Modern: Menggunakan Material Design 3 dengan tema warna yang cerah dan ikon emoji yang menarik.
✅ Validasi Input: Mencegah input negatif atau kosong untuk memastikan hasil yang valid.
📱 Responsif: Tampilan menyesuaikan dengan berbagai ukuran layar perangkat.
🚀 Performa Tinggi: Dibangun dengan Dart native untuk kecepatan maksimal.
